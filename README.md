# CAPC-host

CAPC Host server for docker environment.

## Installation

```sh
pip install -r requirements.txt
```

## Run

```sh
python capc/main.py
```

## FOR PRODUCTION

```sh
poetry export -f requirements.txt -o requirements.txt --without-hashes
```
